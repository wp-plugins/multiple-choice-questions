	Multiple Choice Questions builder for WordPress

== Description ==

An easy way to create Multiple Choice Questions form builder for your WordPress site, use via shortcode anywhere and unlimited. ajax based submission and result.


### Multiple Choice Questions by http://paratheme.com

* [Plugin Page!&raquo;](* [Live Demo!&raquo;](http://paratheme.com/demo/multiple-choice-questions/))
* [Live Demo!&raquo;](http://paratheme.com/demo/multiple-choice-questions/)

<br />


<strong>Plugin Features</strong><br />

* Fully responsive and mobile ready.
* Unlimited MCQ anywhere.
* Use via short-code.
* Different Theme.
* Ajax based result.
* Result pass or failed.
* Unlimited number or question for each MCQ.



== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Multiple Choice Questions</strong>" activate it.<br />

After activate plugin you will see "MCQ" menu at left side on WordPress dashboard click "New MCQ" and use the options field "MCQ Options"<br />

<br />
<strong>How to use on page or post</strong><br />
When MCQ options setup done please publish MCQ<br />

and then copy shortcode from top of <strong>MCQ Options</strong> `[mcq  id="1234" ]`<br />

then paste this shortcode anywhere in your page to display MCQ<br />
